Details

  • A0: Warm-Up (formative)

    This formative assignment is provided for you to get used to the assigment format and assessment process. Instructions on how to do the assignment are in the notebook file, which is also the file you work on and submit for assessment. A tester file is also provided which enables you to test your code while working on it in the notebook.

    • Notebook: A0_Warm_Up.ipynb [View] [Download]
    • Tester: A0_Warm_Up_tests.py [Download]

    To get used to the assignment submission process, you can also submit your A0_Warm_Up.ipynb file via gradescope. There is a link to the Gradescope submission web site on the "Submit my work" page of the module Minerva pages.

    You can submit as many times as you like. If you go to your submission on Gradescope you will see a "re-submit" button at the bottom right of the page.

    Since this piece of work is formative it does not count towards the final grade you will get for this module.

  • A1: Python Programming

    This is the first assignment which is worth 20% of your final module grade. The format is very similar to the formative Warm-Up assignment. Details and files will be released in week 2.

    • Notebook: A1_P4DS2_Python_Programming.ipynb [View] [Download]
    • Tester: A1_P4DS2_Python_Programming_tests.py [Download]

  • A2: Data and Algorithms

    The second assigment is also based on a Jupyter notebook template file with a specification of functions you need to encode. It will involve processing and extracting information from a large dataset.

    • Notebook: A2_Earthquakes_V2.ipynb [View] [Download]

    Note: Tests are not provided for this assignment. You will need to ensure that your code works as specified. However, you will have a 48-hour submission window, within which you can revise and resubmit your code.

  • A3: Data Analysis Project

    A small programming project which is to follow a specified framework and be done within the given template file. You have freedom to choose your own data an aims as long as it fits within the general framework. This can be done individually or in a group of 2 or 3 people.

    • Notebook: A3_Data_Analysis_Project.ipynb [View] [Download]

  • Online Test
    The test will cover all aspects of the module in the form of general questions about programming and data and questions regarding given code examples. The test will be on gradescope.
